Metadata Editor - Northwestern University Library
=========

Server configuration:
- 1-4 Core CPU
- 4GB RAM
- 50GB storage
- RHEL
- nginx
- Passenger
- RVM
- Ruby 2.1
- Git
- mySQL

You'll need to install ImageMagick as well. The easiest way to use homebrew: brew install --with-libtiff imagemagick
